# BERK v0.9.4 Release Binaries

## What's Included

This release contains three LLVM-enabled binaries for Windows x64:

### 1. berk-lang.exe (83.58 MB)
Main BERK compiler with full LLVM backend support.

**Features:**
- Parse and compile BERK source code
- Generate LLVM IR (`.ll` files)
- Generate native assembly (`.s` files)
- Generate object files (`.o` files)
- Generate native executables (`.exe` files)
- Multiple optimization levels (O0, O1, O2, O3)
- Cross-compilation support

**Usage:**
```powershell
# Compile to executable
.\berk-lang.exe program.berk --emit exe -o program.exe

# Generate LLVM IR
.\berk-lang.exe program.berk --emit llvm-ir -o program.ll

# Generate assembly
.\berk-lang.exe program.berk --emit asm -o program.s

# With optimization
.\berk-lang.exe program.berk --emit exe -O 3 -o program.exe
```

### 2. berk-lsp.exe (10.42 MB)
Language Server Protocol server for IDE integration.

**Features:**
- 9 LSP capabilities implemented
- Real-time diagnostics
- Code completion (auto-complete)
- Hover information
- Go to definition
- Find references
- Signature help
- Document symbols
- Code actions
- Semantic highlighting
- Inlay hints
- Code lens

**Usage:**
```powershell
# Start LSP server (stdio mode)
.\berk-lsp.exe

# Used automatically by VS Code extension
```

### 3. berk-repl.exe (9.3 MB)
Interactive REPL for BERK language.

**Features:**
- Execute BERK code interactively
- Immediate feedback
- Variable inspection
- Function testing
- Multi-line input support

**Usage:**
```powershell
# Start REPL
.\berk-repl.exe

# Then type BERK code:
> let x = 42
> yaz("Hello: ", x)
```

## System Requirements

- **OS:** Windows 10/11 (64-bit)
- **Architecture:** x86_64
- **Runtime:** No additional dependencies required (statically linked)

## Build Information

- **BERK Version:** 0.9.4
- **LLVM Version:** 17.0.6
- **Compiler:** rustc 1.83 (stable)
- **Build Profile:** Release (optimized)
- **Build Time:** 55.48 seconds

## Performance

Native executables generated by `berk-lang.exe` run **10-100x faster** than interpreted code, depending on workload.

## Installation

1. Download all three `.exe` files
2. Place them in a directory (e.g., `C:\berk\`)
3. Add directory to PATH (optional):
   ```powershell
   $env:Path += ";C:\berk\"
   ```
4. Verify installation:
   ```powershell
   .\berk-lang.exe --version
   ```
